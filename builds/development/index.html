1st: create basic angular module including a basic controller definition

libs, file references, if there is any controller or service files, do not forget referring them.

use john papa style to create module name and dependencies

    // app.module.js
    angular
        .module('app', ['ngRoute']);

create a {{}} or any other angular feature stuff to test

For example, this is for the test: {{myCtrl.testInit}}
