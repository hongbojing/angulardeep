1st: create basic angular module including a basic controller definition

libs, file references, if there is any controller or service files, do not forget referring them.

as you keep adding different features to the app, remember to assign different controllers to different html parts.

not only use js files to split the app, but also use custome directives (normally, a element directive) to seperate html parts, don't forget to add directive controllers to the directives.

service is also a part of the dependencies, we have some bulit-in services, like $http, and you could define your own services.

use john papa style to create module name and dependencies

    // app.module.js
    angular
        .module('app', ['ngRoute']);

Attach the name of module to our HTML page with a Directive.

create a {{}} or any other angular feature stuff to test

For example, this is for the test: {{myCtrl.testInit}}
